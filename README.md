# Flask Dashboard Monitor Project Documents

This is a Flask web application for Dashboard Monitor. 

## Prerequisites

Before you begin, ensure you have met the following requirements:

- Python 3.x installed on your system.
- [ database setup, API keys, etc.]

## Getting Started

### 1. Create a Virtual Environment

```bash
python3 -m venv .venv
```
### 2. Activate the Virtual Environment
```bash
. .venv/bin/activate
```
###  3. Install Dependencies
```bash
pip install -r packages.txt
```
###  4. Run the Flask Application
```bash
flask run --host=0.0.0.0
```
The application should now be running at http://localhost:5000 or at the specified host and port.
